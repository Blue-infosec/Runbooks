## SMB & Netbios

SMB versions:

* SMB1 – Windows 2000, XP and Windows 2003
* SMB2 – Windows Vista SP1 and Windows 2008
* SMB2.1 – Windows 7 and Windows 2008 R2
* SMB3 – Windows 8 and Windows 2012
